# Normando Restaurant Website

Сайт ресторана "Normando" с интерфейсом в стиле Atrium.

## Структура проекта

- `index.html` - главная страница
- `style.css` - все стили проекта
- `script.js` - интерактивность (попапы, кнопки)
- `assets/` - папка с ресурсами
  - `menu.pdf` - файл меню для скачивания
  - `images/` - изображения
  - `fonts/` - шрифты

## Функционал

- ✅ 4 кнопки в шапке как на wireframe
- ✅ Попап "Окно" с навигацией и телефоном
- ✅ Попап "Меню со свапом" с интерактивными элементами
- ✅ Попап "Корзина"
- ✅ Скачивание меню в PDF
- ✅ Адаптивный дизайн
- ✅ Стиль как на сайте Atrium

## Как запустить

1. Скачайте все файлы
2. Поместите ваш файл меню в `assets/menu.pdf`
3. Откройте `index.html` в браузере

## Кастомизация

- **Логотип**: замените текст в `.logo`
- **Цвета**: измените `#f5b342` на ваш фирменный цвет
- **Шрифт**: в `style.css` можно заменить `Inter` на другой
- **Телефон**: замените номер в `index.html` и `style.css`

## Зависимости

- Google Fonts (Inter)
- Чистый CSS и JavaScript (библиотеки не требуются)
